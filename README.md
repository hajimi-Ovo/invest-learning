## 📝 内容概览

### Fund Financial（基金金融知识）

系统梳理基金投资的核心概念与底层逻辑，涵盖：

- **指数与宽基**：上证指数、沪深300、中证500/1000、标普500、纳斯达克、道琼斯
- **基金类型**：ETF、ETF联接、FOF、QDII、货币基金、指数增强基金
- **核心指标**：夏普比率、卡玛比率、信息比率、Alpha/Beta、最大回撤
- **估值方法**：市盈率（PE）、市净率（PB）、历史分位
- **宏观经济学基础**：利率、通胀、非农数据、流动性、安全边际
- **投资行为**：定投微笑曲线、追涨杀跌的心理陷阱、频繁交易的复利损耗

### 净值系列（2025.06.30）

基金净值的核心概念笔记：

- **单位净值** - 基金当前每份的"单价"，决定买入成本
- **累计净值** - 基金成立以来的总业绩，反映历史赚钱能力
- **分红与净值的关系** - 分红后单位净值强制下降，累计净值不受影响

### 牛奶可乐经济学

《牛奶可乐经济学》读书笔记，提炼书中核心经济学思维：

- 机会成本与成本效益原则
- 边际成本与低果先摘
- 一价定律与套利原理
- 供求模型与市场自我修复

### 纳瓦尔宝典

《纳瓦尔宝典》阅读笔记，聚焦财富与幸福两大主题：

- **三种杠杆**：劳动力杠杆、资本杠杆、"复制边际成本为零的产品"杠杆
- **自我产品化**：专长 × 杠杆 × 判断力
- **幸福哲学**：平和、活在当下、欲望管理
- **投资概念**：安全边际的多种理解视角
