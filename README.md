## 🧠 内容概览（按知识领域）

### 基金与指数投资

系统梳理基金投资的核心概念与底层逻辑，涵盖：

- **指数体系**：上证指数、沪深300、中证500/1000、标普500、纳斯达克、道琼斯
- **基金类型**：ETF、ETF联接、FOF、QDII、货币基金、指数增强、主动管理型
- **核心指标**：夏普比率、卡玛比率、信息比率、Alpha/Beta、最大回撤
- **估值方法**：市盈率（PE）、市净率（PB）、历史分位
- **投资行为**：定投微笑曲线、追涨杀跌、频繁交易的复利损耗
- **基金净值**：单位净值 vs 累计净值，分红与净值的关系

> 📄 核心笔记：`Fund Financial（6.4-6.11）.md`、`20260630_周二.md`

### 期权与衍生品

从零理解期权定价逻辑与策略选择：

- **期权本质**：未来选择权的保险费，买方风险有限、卖方风险无限
- **定价三要素**：内在价值、时间价值、隐含波动率
- **核心关联**：利率（时间成本）↔ 机会成本（决策比较）↔ 时间价值
- **希腊值**：Delta、Gamma、Theta、Vega
- **杠杆效应**：虚值期权的超高杠杆与归零风险
- **期货**：做空机制、合约要素、保证金制度

> 📄 核心笔记：`20260702_周四` → `20260703_周五` → `20260704_周六` → `20260706_周一` → `20260707_周二` → `20260710_周五`

### 宏观经济与货币体系

- **货币派生**：存款创造机制、货币乘数、法定准备金
- **信贷配给**：信息不对称、逆向选择、银行的"安全优先"逻辑
- **财政赤字货币化**：MMT 理论 vs 传统经济学，通胀/通缩风险
- **利率本质**：资金的价格，跨期消费的效用补偿，机会成本的锚
- **通胀机制**：需求拉动、成本推动、货币超发、温和通胀的双刃剑
- **非农数据**：就业 → 收入 → 消费 → 企业利润 → 经济循环

> 📄 核心笔记：`20260717_周五` → `20260718_周六` → `20260709_周四` → `June/Fund Financial`

### 金融底层逻辑与风险

- **奈特不确定性**：风险（概率已知）vs 不确定性（概率未知）
- **风险分配**：保险（锁定生存下限）、衍生品（对冲市场波动）
- **金融本质**：对"稀缺安全"的跨时空分配，将不确定性转化为可定价资产
- **金融市场与个人**：资金的跨期调配、价格信号指导职业与投资

> 📄 核心笔记：`20260721_周二` → `20260722_周三` → `20260723_周四`

### 经济学基础理论

- **机会成本**：五维视角（决策、要素定价、会计-经济、比较优势、动态跨期）
- **成本效益原则**：所有经济学概念的源头
- **边际成本**：低果先摘原则
- **经济租金与寻租**：超额利润的来源与争夺
- **一价定律与套利**：市场自我修复的底层机制

> 📄 核心笔记：`milk_coke_finan.md`、`20260720_周一`、`20260719_周日`、`20260629_周一`

### 技术分析基础

- **K 线形态**：出水芙蓉（看涨反转信号）、影线分析
- **成交量**：量价关系、放量突破与缩量调整
- **技术指标**：均线系统、MACD 等

> 📄 核心笔记：`20260716_周四`、`20260714_周二`

### 阅读笔记
