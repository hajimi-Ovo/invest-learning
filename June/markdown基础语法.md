
下面是**最基础、最常用**的 Markdown 语法，学会这些就能覆盖 90% 的使用场景。

---

### 1. 标题
使用 `#` 符号，有几个 `#` 就代表是几级标题，一共支持 6 级。

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

---

### 2. 段落与换行
- **段落**：直接连续书写文字，段落之间用**空行**隔开。
- **换行**：在行末尾加上**两个或以上空格**，然后回车；或者直接使用 `<br>` 标签。

```markdown
这是第一段。
（这里有一个空行）

这是第二段。

这是第一行末尾有两个空格  
这是换行后的内容。
```

---

### 3. 强调（字体样式）

| 效果         | 语法                  | 示例        |
| :--------- | :------------------ | :-------- |
| **加粗**     | `**文字**` 或 `__文字__` | **粗体**    |
| *斜体*       | `*文字*` 或 `_文字_`     | *斜体*      |
| ***加粗斜体*** | `***文字***`          | ***粗斜体*** |
| ~~删除线~~    | `~~文字~~`            | ~~删除线~~   |

---

### 4. 列表

#### 无序列表
使用 `-`、`+` 或 `*` 作为标记（后面要跟一个空格）。

```markdown
- 苹果
- 香蕉
  - 香蕉的子项目（缩进 2 或 4 个空格）
- 橘子
```

显示效果：
- 苹果
- 香蕉
  - 香蕉的子项目
- 橘子

#### 有序列表
使用数字加英文句点 `.`。

```markdown
1. 第一步
2. 第二步
3. 第三步
```

---

### 5. 链接

- **行内式链接**：`[链接文字](URL "可选的标题")`
```markdown
[百度](https://www.baidu.com)
```
- **链接加标题**（鼠标悬停时显示）：
```markdown
[百度](https://www.baidu.com "点击前往百度")
```

---

### 6. 图片
语法和链接非常像，只是在前面多加一个感叹号 `!`。
```markdown
![替代文字](图片URL "可选的标题")
```

---

### 7. 引用
在段落前面加上 `>` 符号。
```markdown
> 这是一条引用的内容。
> 可以跨多行。
>>
>> 还可以嵌套引用。
```

---

### 8. 代码

- **行内代码**：用反引号 `` ` `` 包裹。
```markdown
使用 `print("Hello")` 函数输出。
```

- **代码块**：用三个反引号 ` ``` ` 包裹，并可以在开头指定编程语言（实现语法高亮）。
````markdown
```python
def hello():
    print("Hello, World!")
```
````

---

### 9. 分割线
使用三个或以上的 `-`、`*` 或 `_` 单独成行。
```markdown
---
***
___
```

---

### 10. 任务列表（待办清单）
常用于 GitHub 或笔记软件。
```markdown
- [x] 已完成的任务
- [ ] 未完成的任务
```

---

### 11. 转义字符
如果你想显示原本有特殊含义的符号（如 `*`、`#` 等），可以在前面加反斜杠 `\`。
```markdown
\* 这不是斜体，而是星号
\# 这不是标题
```

---

### 12.字体颜色
最主流且兼容性最好的方法是使用 `<span>` 标签。你可以像下面这样指定颜色：

```html
<span style="color: 颜色值;">这里是你要变色的文字</span>
```

其中的“颜色值”有几种常用写法：

| 颜色值类型         | 语法示例                                               | 说明                                          |
| ------------- | -------------------------------------------------- | ------------------------------------------- |
| **颜色英文名**     | `<span style="color: red;">红色文字</span>`            | 简单直观，比如 `red`, `blue`, `green`, `orange` 等。 |
| **十六进制代码**    | `<span style="color: #FF0000;">红色文字</span>`        | 最精确，能表示千万种颜色。格式为 `#` 加上6位十六进制数（RRGGBB）。     |
| **RGB/RGBA值** | `<span style="color: rgb(255, 0, 0);">红色文字</span>` | 通过红绿蓝三原色的数值(0-255)来控制颜色，RGBA还可以额外指定透明度。     |

---

### 13.下划线
 使用 `<u>` 标签（最推荐）
 
```markdown
<u>这是带下划线的文字</u>
```

显示效果：<u>这是带下划线的文字</u>
