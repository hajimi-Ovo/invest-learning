# 股票成交量学习笔记：定义、分类与量价关系

> 成交量是技术分析的三大核心要素之一（价格、成交量、时间），与 [[July/20260705_周日|K线基础]] 构成互补的分析框架。本篇系统梳理成交量的定义、放量/缩量特征及核心量价关系规律。

---

## 一、成交量（Volume）的基本定义与本质

### 1.1 定义
成交量是指在特定交易单位时间（如1分钟、1日、1周）内，某只股票（或整个市场）买卖双方达成交易的总手数（1手=100股）。它是市场行为最基础的原始数据之一。

### 1.2 数据本质
成交量的数值直接来源于真金白银的交换，在非对倒交易的常规环境下，其真实性高于技术指标或财务数据修饰。其产生的核心前提是**多空分歧**：只有当买卖双方对当前价格的判断存在差异，且愿意在此价格上完成交换时，成交量才会产生。若市场观点高度一致（一致看涨导致惜售，或一致看跌导致无人承接），成交量则自然萎缩。

> 关于成交量在分时图上的即时表现（红绿柱与主动性买卖盘），可参考 [[July/20260713_周一#三、量能指标：红绿柱状线与成交量柱|量能指标]]。

## 二、放量（Volume Expansion）

### 2.1 定义与特征
放量是指当前周期的成交量相较于此前数个周期（通常对比5日或10日均量线）出现显著放大。放量意味着市场参与资金体量增大，筹码交换剧烈，通常伴随价格趋势的**加速**或**反转**。

### 2.2 关键量化参照
- 若当日成交量达到前5个交易日平均成交量的**2倍**以上，可视为有效放量。
- 若达到**5倍**以上，则属于极端爆量（天量），需高度警惕。

### 2.3 不同价格区间的信号含义

| 价格区间 | 放量形态 | 客观市场含义 |
| :--- | :--- | :--- |
| **长期下跌后的低位区** | 低位放量（阳线） | 长期套牢盘与抄底资金发生剧烈交换。抛压被有效承接，增量资金介入迹象明确，具备底部确认的初步条件。 |
| **上升趋势的中途** | 上涨中继放量 | 新入场者与获利了结者充分换手，市场平均持仓成本上移。健康的放量有助于消化浮筹，为后续上行减轻阻力。 |
| **长期上涨后的高位区** | 高位放量（滞涨/长上影） | 多空分歧达到极致。此时放量可能为主力利用人气旺盛之际进行派发。若放量后股价无法创出新高，则顶部风险显著升高。可结合 [[July/20260707_周二|K线影线形态]] 中的长上影线信号综合判断。 |
| **下跌趋势的中途** | 下跌加速放量 | 恐慌性抛盘与抄底资金短暂博弈。通常为主跌段末期的信号，但未必是绝对底部，需结合后续缩量止跌确认。 |

---

## 三、缩量（Volume Contraction）

### 3.1 定义与特征
缩量是指当前周期成交量明显低于此前数个周期的平均水平。缩量代表市场交投清淡，参与意愿低下，多空双方处于暂时的弱平衡状态。

### 3.2 不同价格区间的信号含义

| 价格区间 | 缩量形态 | 客观市场含义 |
| :--- | :--- | :--- |
| **上升趋势的中途** | 上涨回调缩量 | 下跌过程中缺乏主动卖盘，前期介入资金锁仓意愿较强。此形态通常被定义为**洗盘**，预示主力资金尚未出逃。 |
| **上升趋势的主升段** | 缩量涨停 | 持筹者一致性看多，导致市场卖盘极度稀缺。属于极强的多头延续信号。 |
| **长期上涨后的高位区** | 高位横盘缩量 | 价格维持高位但缺乏新增资金入场。属于**量价背离**，一旦有外部利空打破平衡，容易出现流动性缺失导致的急跌。关于量价背离的详细分类，可参考 [[July/20260711_周六#四、量价配合的几种典型情形|量价配合的典型情形]]。 |
| **下降趋势的中途** | 下跌阴跌缩量 | 此形态极为危险。缩量并非"跌不动"，而是**缺乏承接盘**。由于没有买家，极小的卖盘即可导致价格持续下行，筑底过程将极为漫长。 |
| **长期下跌后的低位区** | 地量（阶段冷清） | 市场情绪降至冰点，主动性抛盘枯竭。是"地量见地价"的必要条件，但非充分条件，需等待后续放量确认底部成立。 |

## 四、核心量价关系客观规律

成交量本身不产生多空指向，必须与价格（K线形态）结合分析。以下是经过市场长期验证的客观量价规律：

1. **量增价升（顺势确认）**：在上升通道初期或中途，成交量温和放大配合价格创出新高。表示买盘力量真实且持续，趋势得到确认。
2. **量增价平（多空转折）**：价格在一个狭窄区间波动，但成交量明显放大。通常出现在底部吸筹末期（主力压价吸筹）或高位筑顶初期（主力缓步派发），后续大概率面临方向选择。
3. **量增价跌（抛压沉重）**：放量下跌通常出现在下降趋势初期或加速阶段。表明市场出现一致性恐慌或主力不计成本出货，应视为风险回避信号。
4. **量平价升（筹码锁定）**：价格持续上涨但成交量维持平稳，未见明显放大。说明上方抛压较轻，主要持筹者目标一致，后市仍有上行惯性。
5. **价跌量缩（趋势延续）**：在下跌趋势中，量能持续萎缩。如前所述，此为"阴跌"特征，代表价格下跌并未吸引到承接盘，原有下降趋势将继续维持。

> 以上量价规律在期货市场中同样适用，但需额外结合 [[July/20260706_周一#七、期货三因素：成交量、持仓量、价格|期货三因素]] 中的持仓量变化综合判断趋势强弱。

## 五、成交量研判的注意事项

- **绝对数值的局限性**：成交量的大小需与**历史均量**及**流通盘大小**对比。同等成交量，在小盘股中可能属于爆量，在大盘股中可能仅属于温和放量。关于流通市值的概念，可参考 [[July/20260713_周一#四、辅助信息面板|辅助信息面板]]。
- **对倒行为的干扰**：主力资金可通过多个账户进行自买自卖（对倒）来人为制造放量假象。因此，**异常突兀的单日爆量**（尤其在小盘题材股中）存在一定失真可能，需结合次日的量能持续性进行综合判断。
- **周期侧重**：日线级别成交量用于判断中短期趋势，周线、月线级别成交量用于判断大周期底部与顶部区域，其可靠程度高于分时级别。
