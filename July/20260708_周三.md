# 投资基本功笔记（一）：K线语言与现金管理工具

## 笔记1：如何正确理解"长上影线"？

> **核心困惑：** 价格明明能涨很高才被压下来，为什么反而说"阻力大"？
> **核心答案：** 阻力大小不看"涨了多少"，而看"被砸回来多少"。

### 1. 形象化理解

| 维度 | 上影线顶端 | 收盘价 |
| :--- | :--- | :--- |
| **含义** | 多头（买方）盘中**摸到**的最高价 | 多头最终**站稳**的价位 |
| **关键指标** | 进攻意愿 | 防守结果 |

- **跳高比赛比喻**：运动员摸到3米高（涨得高），但落地时跌回1米（收盘低）。摔下来的距离（上影线长度）证明了空头（卖方）力量之猛，而不是阻力小。
- **攻城战比喻**：多头冲上城墙（涨得高），但被守军（空头）用巨石砸回城下（跌回原处）。被打回来的距离，就是阻力的直接证据。

> 本篇是 [[July/20260707_周二|影线形态]] 中"长上影线"的深入拆解，配合 [[July/20260705_周日|K线基础]] 中的基础概念一起理解效果更佳。

### 2. 核心逻辑结论

- **阻力小的表现：** 价格涨上去后能**守住**，形成光头阳线或短上影线（涨10块，回撤1块）。
- **阻力大的表现：** 价格被急速拉升后遭遇巨量抛压，自由落体回落，形成长上影线（涨10块，回撤9块）。
- **本质：** 长上影线是"多空力量瞬间逆转"的标志，说明卖方在该价位足以扭转局势。

### 3. 实战位置决定性质

| 出现位置 | 俗称 | 含义与操作参考 |
| :--- | :--- | :--- |
| **连续大涨后的高位** | 射击之星 / 墓碑线 | **强烈见顶信号**。多头力竭守不住，后续动能枯竭，需警惕回调。 |
| **连续大跌后的低位** | 仙人指路 / 试盘线 | **可能为试盘信号**。主力拉高测试上方抛压，若抛压不大（量能未放大），后续反而可能真正突破。 |

### 4. 分析检查清单

下次遇到长上影线，问自己两个问题：
1. 这根K线出现在**山顶**还是**山脚**？（判断多空性质）
2. 被砸回来的那一段，**成交量**是否突然放大？（放量=抛压真实；缩量=可能洗盘）

## 笔记2：国债逆回购——股市"零钱理财神器"

> **一句话定义：** 逆回购，就是**"用国债做抵押，把钱借给别人，到期收本金+利息"**的超短期无风险理财。

### 1. 概念拆解与角色定位

- **你的角色：** 资金出借方（债主），赚取利息
- **对手方角色：** 持有国债的金融机构（银行/券商），支付利息
- **抵押品：** 国债（国家信用背书，风险极低）
- **安全性：** 极高，几乎等同于国债（除非国家破产）

> 逆回购的利率本质上就是 [[July/20260703_周五|无风险利率]] 的一种市场表现形式。

### 2. 生活化案例

朋友（机构）拿价值10万的"国债券"押给你，找你借10万现金用1天，承诺明天还你10万零5块利息。你有抵押品在手，觉得划算且安全，于是成交。
→ **你** = 逆回购方；**朋友** = 正回购方。

### 3. 实操指南（股票软件操作）

| 项目 | 具体说明 |
| :--- | :--- |
| **操作入口** | 股票交易软件 → 找到"国债逆回购"或输入代码 |
| **操作方向** | 点击 **"卖出"**（重要！因为是卖出资金使用权，不是买入） |
| **常见品种** | 沪市：GC001（1天期）、GC007（7天期）；深市：R-001、R-007 |
| **投资门槛** | 沪深两市均**1000元**起（深市原1000元，沪市原10万现已降至1000元） |
| **资金占用** | 买1天期（GC001），**第二天开盘前**本金利息自动到账，**不影响当日买股票** |

### 4. 最佳操作时机（利息飙升时点）

当市场"缺钱"时，利率会大幅走高，此时操作最划算：
- **时间节点：** 月末、季末、年末（银行考核）
- **长假前夕：** 国庆、春节等（资金需求大）
- 此时年化利率常可达到 **5%–10%**，远超普通货币基金（余额宝）

### 5. 角色对比总结

| 对比维度 | 买股票 | 做国债逆回购 |
| :--- | :--- | :--- |
| **目的** | 赚取差价（高风险） | 赚取固定利息（零风险） |
| **收益** | 不确定，可能亏损 | 固定，约定多少就多少 |
| **流动性** | T+1卖出，T+2取现 | 1天期次日资金可用，流动性极强 |
| **用途** | 资产增值 | 打理账户**闲置过夜资金** |

> **每日操作心法：** 每天下午3点收盘后，如果账户有闲钱且不买股票，做一个1天期逆回购（GC001），利息远高于活期（0.3%），第二天开盘钱又回来了，**既不耽误炒股，又能吃利息**。
